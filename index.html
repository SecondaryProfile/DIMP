DIMP

Digital Icon Maker & Painter

A web-based icon design tool built for speed and simplicity. Draw icons directly in your browser with vector shapes, paths, and text — no installs required.

Features

  • Vector shapes: line, circle, square, triangle, path
  • Bezier curve editing on paths
  • Eraser and eyedropper tools
  • Export: SVG, PNG, JPG, WebP, iOS AppIcon.appiconset, Android mipmap bundle
  • Color palettes, grid snapping, undo/redo
  • Responsive — works on desktop and mobile

Built with HTML5 Canvas · Pure JavaScript · No frameworks
Source: github.com/SecondaryProfile/DIMP
